# Evercrew Local Telegram Task Bot Roadmap

## Vision

Build a free, open-source, local-first AI workflow assistant for SMEs.

The project demonstrates practical AI automation using small local LLMs that can run on ordinary office computers.

---

# Version 1.0

Goal:

Turn boss instructions into structured employee tasks.

Features:

- Telegram Bot
- Local Ollama support
- Boss confirmation
- Task breakdown
- Employee notification
- DONE reply

---

# Version 1.1

Improve task understanding.

Features:

- Better deadline extraction
- Better employee name detection
- Priority detection
- Multiple task extraction

---

# Version 1.2

Workflow improvements.

Features:

- Task history
- Task status
- Daily summary
- Reminder messages

---

# Version 2.0

Support additional connectors.

Planned:

- WhatsApp
- Email
- Microsoft Teams
- Slack
- Discord

---

# Version 3.0

Evercrew Local Workflow Engine

Separate reusable workflow engine from Telegram connector.

Future projects will reuse the same engine.

Examples:

- Meeting Minutes Assistant
- Invoice Approval Assistant
- Leave Request Assistant
- Customer Enquiry Router
- AI Admin Assistant

---

# Long-Term Goal

Become the open-source local AI workflow platform for SMEs.
