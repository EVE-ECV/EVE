# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful, and inclusive environment for everyone participating in the EVE community.

We encourage constructive discussions, collaboration, and knowledge sharing regardless of experience level, background, or identity.

## Expected Behaviour

Participants are expected to:

- Be respectful and professional.
- Be open to constructive feedback.
- Focus on solving problems together.
- Help new contributors learn.
- Respect differing viewpoints and experiences.

## Unacceptable Behaviour

Examples include:

- Harassment or abusive language.
- Personal attacks.
- Discrimination of any kind.
- Publishing private information without permission.
- Spam or intentionally disruptive behaviour.

## Reporting Issues

If you experience or witness unacceptable behaviour, please contact:

support@evercrew.ai

All reports will be handled respectfully and confidentially whenever possible.

## Scope

This Code of Conduct applies to:

- GitHub repositories
- Discussions
- Issues
- Pull Requests
- Community communication channels
